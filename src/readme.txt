To Run the Demo:

Open up a mongodb server (mongod)
Run db/utils.py to populate the database
Run main.py to start up the Tornado server
Visit localhost:8888 in your browser
Log in as ml2877@columbia.edu, password: admin
Click Intro to CS
Click Events
Click Lecture

You should now be in the Demo for the event Lecture

Try visiting this page in multiple different windows,
